A simple example of a CRUD application in Java using Spring Boot, Hibernate, Thymeleaf and Angular 5.

## Quick start

To run Spring Boot backend:

- Install PostgreSQL and create a database called "filmes"
- Update the password of the database at the file persistence.xml 
- Run App.java as Java Application to start Spring Boot

To run Thymeleaf frontend:
- Insert in your browser the URL http://localhost:8080/filmes

To run Angular 5 frontend:
- npm install -g @angular/cli
- ng new client-angular-5
- cd client-angular-5
- Replace src folder 
- ng serve --open (will open at http://localhost:4200)
